# XMas Chall

## Get Started
- Install rust `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
- Install deno `curl -fsSL https://deno.land/x/install/install.sh | sh`

## Run
- To run application `make ip-web`
- To run tests `make ip-curl`
